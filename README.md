# E-Commerce Project (react-project-test)

Un proyecto de E-Commerce desarrollado con tecnologías modernas, enfocado en rendimiento y una experiencia de usuario fluida.

## 🚀 Stack Tecnológico

- **Framework:** [Next.js](https://nextjs.org/) (Versión 16)
- **Librería UI:** [React](https://react.dev/) (Versión 19)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/) (Versión 4)
- **Base de Datos & Autenticación:** [Supabase](https://supabase.com/)
- **Pagos:** [Mercado Pago](https://www.mercadopago.com/)
- **Mails:** [Resend](https://resend.com/)

## 📦 Dependencias Principales

- `@mercadopago/sdk-react` y `mercadopago` para la integración de pagos.
- `@supabase/ssr` y `@supabase/supabase-js` para interacciones con el backend.
- `react-icons` para la iconografía.
- `resend` para el envío de correos electrónicos.

## 💻 Instalación y Uso

1. Clonar el repositorio.
2. Ejecutar `npm install` para instalar las dependencias.
3. Configurar las variables de entorno necesarias (Supabase, Mercado Pago, etc.).
4. Ejecutar `npm run dev` para iniciar el servidor de desarrollo.
